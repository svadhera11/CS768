About

This course covers machine learning methods that operate on graph-structured and set-structured data. Topics include graph neural networks, combinatorial optimization, link prediction, information cascades, DeepSets, set distances, and locality sensitive hashing — with an emphasis on reading and implementing recent research papers.

Prerequisites

No formal prerequisites, but familiarity with linear algebra, discrete mathematics, probability, and basic machine learning is helpful. For projects, familiarity with Python and PyTorch would also be useful.

Syllabus

Tentative. Topics subject to change.

  • Graph Neural Networks
  • Graph Distances
  • Combinatorial Algorithms and GNN VC, IS, MaxClique, TSP  ·  Neural Algorithmic Reasoning  ·  Graph Matching (Optimal Transport)
  • Link Prediction
  • Information Cascades
  • DeepSets
  • Set Distances
  • Locality Sensitive Hashing (LSH)
