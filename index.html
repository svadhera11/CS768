About

This course covers machine learning methods that operate on graph-structured and set-structured data. Topics include graph neural networks, combinatorial optimization, link prediction, information cascades, DeepSets, set distances, and locality sensitive hashing — with an emphasis on reading and implementing recent research papers.

Prerequisites

Linear algebra, probability, and basic machine learning (CS 419 or equivalent). Familiarity with Python and PyTorch is expected.
